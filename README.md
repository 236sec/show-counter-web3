## Getting Started

### Deploy Contract and Compiled to get ABI

1. Deploy Contract and get the Contract Address to replace in `lib/counter.ts`
2. Config the network that you want to connect in `config.ts`

### Run the development server:

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
